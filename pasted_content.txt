## **📝 Arquivo Modificado:**

### **3. `netlify.toml` (ALTERADO)**
**ANTES:**
```toml
[build]
  publish = "dist"
  command = "pnpm run build"  # ❌ PROBLEMA: pnpm não disponível
```

**DEPOIS:**
```toml
[build]
  publish = "dist"
  command = "npm run build"   # ✅ CORRIGIDO: usando npm
```

**Mudança específica:**
- ❌ **Removido:** `pnpm run build`
- ✅ **Adicionado:** `npm run build`
